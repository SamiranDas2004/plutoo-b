## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Visit `http://localhost:5173` to view the website.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

### Colors

The main brand colors are defined using CSS gradients:
- Primary: `#6366f1` (Indigo)
- Secondary: `#8b5cf6` (Purple)
- Accent: `#ec4899` (Pink)

### Content

Edit the component files in `src/components/` to customize:
- Features list in `Features.vue`
- Pricing plans in `Pricing.vue`
- Testimonials in `Testimonials.vue`
- Navigation links in `Navbar.vue`
